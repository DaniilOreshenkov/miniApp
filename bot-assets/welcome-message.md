# Приветственное сообщение /start

## Вариант 1 — Лаконичный (рекомендуется)

```
Привет 👋

Это **beadly** — твой мини-апп в Telegram.

Нажми кнопку ниже, чтобы начать.
```

**Кнопки (inline keyboard):**
| Текст | Действие |
|---|---|
| Открыть beadly 🚀 | [URL кнопка → ссылка на мини-апп] |

---

## Вариант 2 — С описанием

```
Привет, {имя} 👋

Добро пожаловать в **beadly**.

Здесь всё просто:
• Открываешь приложение
• Делаешь что нужно
• Profit

Готов начать?
```

**Кнопки:**
| Текст | Действие |
|---|---|
| Открыть приложение | [URL → мини-апп] |
| Помощь | /help |

**Кнопки:**
| Текст | Действие |
|---|---|
| Открыть beadly | [URL → мини-апп] |

## Советы по оформлению сообщений

- Отправлять баннер `banner-welcome.svg` (PNG) + текст первым сообщением — выглядит профессионально
- Использовать `parse_mode: HTML` или `MarkdownV2`
- Кнопки — `InlineKeyboardMarkup` с типом `web_app`
