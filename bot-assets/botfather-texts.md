# Тексты для оформления бота в BotFather

## /setname
Beadly

## /setusername
@Beadlybot

---

## /setdescription
(показывается в карточке бота до первого сообщения, до 512 символов)

```
Beadly — приложение для создания схем из бусин прямо в Telegram.

Загрузи любое изображение — и получи готовую схему в один клик. Рисуй вручную, редактируй цвета, сохраняй и делись результатом.

Нажми «Запустить», чтобы начать →
```

---

## /setabouttext
(короткая строка в профиле, до 120 символов)

```
Схемы из бусин в один клик. Загрузи фото — получи готовую схему ✦
```

---

## /setcommands

```
start - Открыть Beadly
help - Как пользоваться
```

---

## /setmenubutton
Настроить кнопку меню (Menu Button) → выбрать «Web App»:
- Текст кнопки: ✦ Открыть Beadly
- URL: https://mini-app-ruddy-sigma.vercel.app

---

## Заметки по настройке

- Аватарка: PNG 512×512, загрузить через /setuserpic
- Description: показывается до первого сообщения (до 512 символов)
- About: короткая строка в профиле (до 120 символов)
- Menu Button: заменяет скрепку на постоянную кнопку открытия приложения — очень удобно
