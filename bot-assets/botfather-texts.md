# Тексты для оформления бота в BotFather

## /setname
beadly

## /setusername
(оставить текущий или: @beadly_bot / @beadlyapp_bot)

---

## /setdescription
(показывается в карточке бота до первого сообщения)

```
beadly — это твой мини-помощник прямо в Telegram.

Быстро, удобно, без лишнего.
Всё что нужно — в одном месте.
```

---

## /setabouttext
(короткая строка в профиле, до 120 символов)

```
Твой мини-апп в Telegram. Открой и попробуй →
```

---

## /setcommands

```
start - Открыть beadly
help - Помощь
```

---

## Заметки по настройке

- Аватарка: загрузить `avatar.svg` (конвертировать в PNG 512×512) через /setuserpic
- Описание (about): до 120 символов
- Description: до 512 символов, показывается до первого сообщения
