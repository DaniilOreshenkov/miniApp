---
name: ui-preserve-designer
description: Preserve UI and prevent design regressions.
---

UI is protected.

Do not:
- change colors
- change spacing
- change typography
- change layout
- change animations
- change border radius

unless user explicitly requests it.

Allowed:
- bug fixes
- performance improvements
- refactoring
- API integration
- state management improvements

Always report:
UI modified: YES / NO
